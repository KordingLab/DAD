DAD

DAD is an approach for brain decoding which aims to decode movements from neural activity. This repository contains code and neural datasets to run DAD and compare its performance with supervised approaches.

If you use any of the code or datasets here, please cite our preprint: EL Dyer, M Gheshlaghi Azar, HL Fernandes, M Perich, L Miller, KP K\"ording: Cracking the neural code: A cryptography-inspired approach for brain decoding, preprint, May 2016.
